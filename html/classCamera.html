Detailed Description

This class represents a camera object that will take a picture of the 3d environment and represent this as an image.

This class handles all the projection and drawing necessary to take a picture.

In order to facilitate this, the object needs to be added to a stage, the stage contains all the vectors and surfaces this means the camera must be added to it so it can take the relavant picture.
